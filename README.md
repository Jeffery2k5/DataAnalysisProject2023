The data used for this project is the weapons and operators CSV files in Rainbow six siege as of 2023. This data set was used in my first python data analysis project along with my first time using github. The main goal of this project was learn skills in order to develop a data analysis system that could detect patterns, trends and key insights given a CSV dataset. The intention of this project was to help develop skills necessary to use for data analysis along with learning how to use a Github enviornment.
